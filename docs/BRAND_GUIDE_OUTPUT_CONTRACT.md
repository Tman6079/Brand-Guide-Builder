# Brand Guide Output Contract

All brand guides generated by this application MUST conform to the following structure and intent.

This structure is invariant across all input modes (URL scan, Battle mode, Survey).

The guide is instructional, intended for both AI systems and human writers.

Sections must appear in the following order:

1. Brand Essence & Mission
2. Voice & Tone Rules
3. Messaging Framework (SPICED)
4. Headline & CTA Guidelines
5. Performance Benchmarks & Data Reference
6. End Rules
